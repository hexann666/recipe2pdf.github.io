Requirements

Requirements Release 1:

  • System must show an input field.
  • User should insert a web-link into this field.
  • System scrapes web content from the provided link and should look for following informations: recipe_title, ingredients, instructions, notes.
  • If recipe_title, ingredients, instructions were found, system must generate a pdf file with recipe_title, ingredients, instructions, notes (if found) according to the template specified in the model configurations after maximum 20 s.
  • System must show pdf in a new tab.

Requirements Release 2:

  • User can tick a checkbox with text “May we store this recipe in our data base?” (consent checkbox), checkbox is optional for the flow.
  • If user gave consent ticking the checkbox and- If recipe_title, ingredients, instructions were found, system must store the generated pdf file and recipe_title, ingredients, instructions, notes (if found) in the data base with th same key.
  • Information in the database shall be reachable, if user performs a search and shall provide result with responce time of maximum 30 s.

Requirements Release 3:

  • Provide a choice of templates as pictures (jpg-files) below the insert field for a link
  • If user clicks on a template picture, show the picture in front of the current webpage. When User clicks on any part of the picture, hide the picture.
  • If user doesn’t choose a template, use a default template to generate PDF from her link

Requirements Release 4:

  • System must show an input field for ingridients.
  • User should type in ingridients separated by comma.
  • After typing in the ingridients, user shall press button “inspire me!” to confirm.
  • After user pressed “inspire me!” button, validate the input. Only following characters are allowed [A-Za-z, ]
    • If any special character are typed in, don’t proceed to the next step and show the error message below the input field: “No special characters are allowed.”
    • if validation succeeded, use input as input in the script for ML script.
  • ML script shall be ready in max 40 s.
  • System should show a loading animation while ML script is running.
  • As soon as ML script is ready, the system should show the title of the result with highest score below the “inspire me!” button. The title is a hyperlink to the PDF of the recipe.
  • If user clicks on the recipe title, system should show a PDF of this recipe in the new browser tab.
